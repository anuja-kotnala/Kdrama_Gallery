## K-Drama Gallery 🎬
An interactive cinematic K-Drama gallery built using HTML, CSS, and JavaScript.
Hover over drama titles to reveal floating scene previews with smooth animations and dynamic theme changes inspired by each drama’s atmosphere.

## ✨ Features
Interactive hover-based UI
Floating cinematic preview images
Dynamic theme/background changes
Smooth transitions and animations
Minimal aesthetic design
Responsive centered layout
Pure HTML, CSS, and JavaScript  

## 🎭 Dramas Included
Crash Landing on You
Taxi Driver
Vincenzo
Hospital Playlist
Yumi's Cells

## 🖼️ Preview

When a user hovers over a drama title:

3 themed scene images appear around the text
images smoothly pop into position
page colors change according to the drama vibe
cinematic blur and transition effects are triggered
